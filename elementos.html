Elementos Semánticos del texto

Elementos - Resaltado

- ITALIC I: Cursiva

Esto es un texto en cursiva.

- BOLD B: Negrita

Esto es un texto en negrita.

- UNDERLINED U: Subrayado

Esto es un texto subrayado.

				Cursiva
	<p><i>Esto es un texto en cursiva.</i></p>
    
    			Negrita
	<p><b>Esto es un texto en negrita.</b></p>
    
    			Subrayado
	<p><u>Esto es un texto en subrayado.</u></p>

- STRESS EMPHASIS EM: Recalca parte del texto o le pone énfasis.

Esto es un texto con énfasis.

- STRONG: Parte seria o importante.

Esto es un texto remarcado.

- SMALL: Representa la letra pequeña.

Esto es un texto con letra pequeña.

- S: Se usa para tachar lo que ya está obsoleto o ya no vale.

Esto es un texto que ya no vale.

				Énfasis
	<p><em>Esto es un texto con énfasis.</em></p>
    
    			Strong
	<p><strong>Esto es un texto remarcado.</strong></p>
    
    			Small
	<p><small>Esto es un texto con letra pequeña.</small></p>
    			
               Tachado
	<p><s>Esto es un texto que ya no vale.</s></p>

Elementos - Salto de línea

BREAK: BR Hace un salto de línea.

WORD BREAK: WBR

                Salto de línea
	<p>Esto es un salto de línea.</p><br>

Elementos - Hiperenlace

ANCHOR: a Hace un salto de línea.

- Enlace a URL: href

Marca un lugar en el que podría haber un enlace si fuera relevante.

Este texto contiene un enlace a la página de Google Google

                Hiperenlace
	<p>Este texto contiene un enlace a la página de Google.
    <a href="www.google.es">Google</a></p>

Atributos (si usamos href)

  • download
    • -Valor opcional: nombre en destino.
  • target
    • - _blank
  • otros

Elementos - Hiperenlace interno

         Identificador de Elemento
	<elto id="eltoX">Elemento X</elto>
    
         Referencia a Elemento
	<a href="#eltoX">Elemento X</a>
    
         Principio del Documento: #top

Elementos - Hiperenlace general

Referencia desde otro documento: Debo indicar cual es la URL del documento y concatenar a la misma el identificador de ese lugar.
