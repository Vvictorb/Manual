Agrupación - Listas

Lista Ordenada

  1. Cosa 1
  2. Cosa 2
  3. Cosa 3
                    Lista Ordenada
        	
    <ol>
  <li>Cosa 1<li>
  <li>Cosa 2<li>
  <li>Cosa 3<li>
</ol>

Argumentos de la lista Ordenada:

  • reversed
  • start="número"
  • type="tipo"
    • - l
    - a
    - A
    - i
    - I
  • Para LI: value

Lista Desordenada

  • Cosa 3

  <li>Cosa 1<li>

Agrupación - Línea Horizontal

HORIZONTAL RULE: HR

La siguiente línea horizontal divide temáticamente este párrafo del siguiente.

Este segundo párrafo puede pertenecer,por ejemplo, a un cambio de escena dentro de un mismo capítulo de una novela

                    Línea Horizontal
                    
<p>La siguiente línea horizontal divide temáticamente este párrafo del siguiente.</p>
<hr>
<p>Este segundo párrafo puede pertenecer,por ejemplo, a un cambio de escena dentro de un mismo capítulo 
de una novela.</p>
